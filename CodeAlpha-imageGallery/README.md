# Image Gallery

A responsive image gallery built with HTML, CSS, and JavaScript featuring navigation, lightbox view, hover effects, smooth transitions, and image filters.

## Features

- **Responsive Design**: Adapts to different screen sizes using CSS Grid.
- **Lightbox View**: Click on any image to view it in a full-screen modal.
- **Navigation**: Use next/prev buttons to navigate through images in the lightbox.
- **Hover Effects**: Images scale and fade on hover.
- **Smooth Transitions**: All animations use CSS transitions for smooth effects.
- **Image Filters**: Filter images by categories (All, Nature, City, Animals).

## Usage

1. Open `index.html` in a web browser.
2. Click on filter buttons to show images from specific categories.
3. Click on any image to open the lightbox view.
4. Use the arrow buttons or close button to navigate or exit the lightbox.

## Files

- `index.html`: Main HTML structure.
- `styles.css`: CSS for layout, responsiveness, and animations.
- `script.js`: JavaScript for interactivity (lightbox, navigation, filters).

## Customization

- Replace placeholder images in `index.html` with your own images.
- Update categories in the `data-category` attributes.
- Modify styles in `styles.css` for different themes.

## Troubleshooting

- Ensure all files are in the same directory.
- If images don't load, check internet connection (placeholders are from Lorem Picsum).
- For local images, use relative paths in `src` attributes.
